# Markdown

[Markdown Cheat Sheet \| Markdown
Guide](https://www.markdownguide.org/cheat-sheet/)

### Dropdowns

``` markdown
<details>
	<summary>Topic</summary>
	This text will be hidden.
</details>
```

### Table of content TOC

``` markdown
Table of Contents
=================
  * [Topic](#class-name)
  <a href="class-name">Topic</a>
```

### Empty line break `<br/>`


### Italic/ bold/ header

``` markdown
<!--italic-->*Hello*
<!--bold-->**Hello**
<!--h3-->### Hello ###
```
