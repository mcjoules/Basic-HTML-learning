### Dropdowns

```markdown
<details>
	<summary>Topic</summary>
	This text will be hidden.
</details>
```



### Table of content TOC

```markdown
Table of Contents
=================
  * [Topic](#class-name)
  <a href="class-name">Topic</a>
```



### Empty line break `<br/>`

```html
<!--sometimes an empty line break is required, so <br/> does not affect other syntax/elements in markdown-->
<br/>
<br/>
```



### Italic/ bold/ header

```markdown
<!--italic-->*Hello*
<!--bold-->**Hello**
<!--h3-->### Hello ###
```
